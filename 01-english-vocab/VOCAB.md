## Symbols & Punctuation
| English | 中文 | 符号 | example usage |
|--------|------|------|-----------|
| parentheses | 小括号 | `( )` | function call, condition |
| curly braces / curly brackets | 大括号 | `{ }` | dict, code block, scope |
| square brackets | 中括号 | `[ ]` | list, index, array |
| colon | 冒号 | `:` | if/for/function definition |
| semicolon | 分号 | `;` | end a statement |
| single quote | 单引号 | `' '` | string |
| double quote | 双引号 | `" "` | string |
| open / close quotation marks | 开/闭引号 | ` " " ` | string |
| hash / pound sign | 井号 | `#` | comment |
| underscore | 下划线 | `_` | variable naming |
| dot / period | 点 | `.` | attribute, method |
| comma | 逗号 | `,` | separate parameters |
| backslash | 反斜杠 | `\` | escape character |

## Python Basic Terms
| English      | 中文 | Explanation |
|--------------|------|-------------|
| variable     | 变量 | stores a value in memory |
| function     | 函数 | reusable block of code |
| define       | 定义 | create a function with def |
| parameter    | 形参 | input placeholder in a function |
| argument     | 实参 | actual value passed to a function |
| return       | 返回 | send a result back from function |
| import       | 导入 | load external modules or libraries |
| iterate      | 遍历 | loop over items in a collection |
| loop         | 循环 | for loop and while loop |
| condition    | 条件 | logic in if/elif/else |
| list         | 列表 | ordered and mutable collection |
| dictionary   | 字典 | key-value storage structure |
| index        | 索引 | position in a list or array |
| assign       | 赋值 | set value to a variable |
| initialize   | 初始化 | set an initial value |
| execute      | 执行 | run a piece of code |
| syntax       | 语法 | rules for writing valid code |
| debug        | 调试 | find and fix errors in code |
| output       | 输出 | result printed or returned |
| input        | 输入 | data provided by user or file |

- We implement a simple model to predict the target value.
- The dataset contains N samples and multiple features.
- We use the ______ algorithm to achieve the task.
- The model achieves an accuracy of ______ on the test set.
- We preprocess the data by normalization and cleaning.
- All code is implemented in Python using popular ML libraries.
